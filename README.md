# Реактивные шахматы

A starter project for using React, TypeScript, SCSS and Webpack.

## Используемые технологии
- Webpack 5 + HMR
- TypeScript + React 18
- SCSS + Autoprefixing 

## Установка и запуск

1. Clone the project
2. Run `npm install`
3. Run `npm run dev`
